# Task 5.2

## Description

Write a program that empirically checks if the following methods are interruptible:
- `Thread.sleep`
- `ReentrantLock.lock`
- `monitor_enter` in `synchronized`
- `Condition.await`
- `Object.wait`
- `System.in.read`
- `FileWriter.write`

## Requirements

Ensure your answer is clear, readable and concise.
